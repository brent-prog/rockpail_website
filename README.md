# RockPail Website

Marketing website for RockPail.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- Vercel-ready deployment

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build check

```bash
npm run build
npm run lint
```

## Current site scope

This is the first landing-page version:

- Hero: Keep Fun Simple
- What is RockPail?
- How to Play: Fill, Place, Pace, Play
- Why It Works
- Rules Preview
- FAQ
- Contact / early inquiry CTA
